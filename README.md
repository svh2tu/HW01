# HW01
This is the first repository that we will use for HW this semester. 

1) Clone the repository to your area on Rivanna (like we learned in class02)
2) Complete the suite of Linux tutorials found here: https://learning.rc.virginia.edu/notes/unix-tutorial/
3) Take time to understand what each command is doing.  Why might that be useful?  
4) When done run the command:
>>> history 100 > history.txt
#Your last 100 commands should go into this file
5) then add history.txt to your repo, commit, and push your changes
6) Verify on GitHub that your history.txt file push was successful.
7) You are done! Add history.txt to the HW01 assignment on Canvas.
